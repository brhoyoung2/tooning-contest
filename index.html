DIVISION 01
만화 부문Comics / Webtoon
  • 10컷 이상 (표지 포함)10+ panels (incl. cover)
  • 해상도 1080 × 1080 pxResolution 1080 × 1080 px
  • 3종 세트 제출 — 이미지 + 설명 + 에디터 링크3-item set — image + description + editor link
DIVISION 02
소설 부문Novel / Story
  • 1화당 2,000자 이상2,000+ characters per episode

  • 소설 3화 분량 제출Submit 3 episodes
DIVISION 03
카드뉴스 부문Card News
  • 10컷 이상 (표지 포함)10+ panels (incl. cover)
  • 정보 전달형 콘텐츠 — 이슈·교과·시사Informational content — issues, curriculum, current events
  • 이미지 + 설명 + 에디터 링크 제출Image + description + editor link
